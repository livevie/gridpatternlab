component/multicolumn elements Responsive multicolumn elements

Component Description

With the multicolumn elements component, the order of the HTML elements and how they are displayed plays an important role. With display: flex or display: inline-block it's possible to output multicolumn elements with a fixed width but these are displayed horizontally and not vertically. In addition, it's not possible to directly determine how many columns may be created.

With CSS Grid both rows and columns can be defined. With grid-template-columns the size and number of a column can be defined and with grid-template-rows the size and number of a row can be defined. grid-auto-flow additionally controls how auto-placed placed elements get flowed into the grid.
