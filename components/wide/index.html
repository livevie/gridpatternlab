component/content dependent column Column as wide as widest child item content

Component Description

Setting content-dependent columns is only possible with css grid. display: flex or float gives you the possibility to create multi-column layouts, but these are always defined with a fixed width and are not flexible according to content.

By creating grid-template-columns you can precisely define the flexibility of a grid layout and adjust it to your needs. max-content specifies the maximum size the content needs without wrapping the text
