Component Description

The article component is one which is very often used on websites. It can also be created with display: flex or float, but it's difficult to display the image and text in exactly the same size and with a space between them, without wrapping the entire content. This requires widths, margins or paddings.

With CSS Grid it's possible to align elements to a grid. In this example the article container is defined in a 12-column grid and the elements are aligned within this grid. grid-gap defines the distance between the elements within the grid.
