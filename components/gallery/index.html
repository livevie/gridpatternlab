component/gallery Responsive image gallery

Component Description

The gallery component should be easy to implement with display: flex or display: inline-block. But it's not. For both solutions an additional margin is required, which has to be the same on all sides to create a distance between the gallery images. However, there is also a mandatory gap at the edges, which is often undesirable. In addition, in the case of inline block elements, they interpret white-space in the markup in a way that introduces extra unwanted space between each item. You can get rid of this extra spaces, but you’ll have to implement one of several workarounds.

CSS Grid has the perfect solution for this problems: adding grid-gap to the component container. This solution creates space only between the individual elements.
