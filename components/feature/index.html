component/feature box Responsive feature boxes

Component Description

Creating a feature box with display: flex or display: inline-block is a little bit tricky. In this component the headline "Feature Section" and the paragraph "Lorem, ipsum dolor ..." should be positioned in one line. Without giving the paragraph a margin or a padding it's impossible. Here you have to work pixel-precisely.

CSS Grid makes it possible to work without such hacks. With a fixed size and the definition of grid-template-columns to the feature container. Here you define the size of the columns in which the elements are located and define the distance to each other with grid-gap. Margins and paddings are no longer needed.
