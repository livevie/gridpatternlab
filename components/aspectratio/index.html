component/aspect ratio boxes Aspect ratio box layout

Component Description

This aspect ratio boxes layout based on equal sized square grid cells, where grid items could span one or two cells on the row and/or column axis. In other words, the grid cells needed to maintain an aspect ratio (1:1 in this case), but the actual grid items did not necessarily.

Doing Components like this involves trickery with padding such that an element's width and height are in proportion to your need, but it's possible with CSS Grid.

Here we’re going to use grid-template-columns and grid-auto-rows to define our grid. grid-auto-rows is a useful alternative to grid-template-rows for defining implicitly created grid tracks for the case we don't know how many rows there will be, but we want them all to be the same height.

In this particular grid we can let CSS Grid’s auto placement algorithm do some of the work. The small grid items should span a single cell and follow the normal flow. The large grid items should span two tracks in either direction.

If you want the rest of the items to fill in any spaces you can use grid-auto-flow: dense; on the grid container, otherwise any subsequent items will simply take the next place in the flow.

I have to support:

Code

  • CSS
  • HTML
<div class="container"> <div class="item item__lg">1</div> <div class="item item__sm">2</div> <div class="item item__sm">3</div> <div class="item item__db">4</div> </div>
